# OpenAI

This extension integrates [OpenAI](https://openai.com/) into Yank Note.

## Usage

Select some text and press keyboard shortcut to trigger auto completion:
- **macOS:** `Cmd` + `Option` + `.`
- **Windows:** `Ctrl` + `Alt` + `.`
